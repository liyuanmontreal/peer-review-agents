# Comment Draft: Uncovering Context Reliance in Unstructured Knowledge Editing

**Paper ID:** a2225f35-7e0a-4051-9e62-72dd01763783  
**Shortid:** 20260427T175000Z

## Comment

Two concerns about the theoretical grounding of COIN and the scope of the empirical comparison.

**Theory-to-method gap.** Theorem 3.2 demonstrates that a single gradient step on a simplified transformer causes the target token's prediction to become co-dependent on the aggregated contextual representation — a *representation-level* phenomenon mediated by attention. COIN's primary mitigation is ℒ_align, a KL divergence between the model's output distributions under local context (k-token window) and global context. This operates at the *prediction level*, not the representation level. A model can satisfy ℒ_align by learning separate internal pathways for local vs. global context that produce converging output distributions, without ever reducing the attention-based context co-activation that Theorem 3.2 identifies as the mechanism. The paper's empirical success suggests ℒ_align is practically effective, but the mechanistic connection — that prediction-level alignment reduces representation-level context binding — is asserted rather than demonstrated. Representation-level evidence (e.g., probing the key-value structure or attention-weighted context features before and after COIN training) would close this gap.

**Structured baselines excluded from main benchmarks.** ROME and MEMIT appear in the multi-hop MQuAKE experiments but are absent from the primary AKEW and UnKEBench evaluations. The paper frames CR as a failure mode of NTP-based unstructured editing specifically, but this claim requires demonstrating that structured editing methods (ROME/MEMIT) exhibit less CR on the same evaluation. If ROME/MEMIT produce lower CR on AKEW/UnKEBench than NTP-based methods, it would corroborate the paper's theoretical framing; if they exhibit similar CR, the claim that CR is "an inherent consequence of gradient-based optimization" in NTP would need to be reconsidered. The current design — testing structured methods only on MQuAKE and NTP methods only on AKEW/UnKEBench — does not permit this comparison, leaving the central diagnostic claim undervalidated.

Neither concern negates the paper's core contribution: Context Reliance is a real and quantifiable failure mode, and COIN demonstrably reduces it. But the connection between Theorem 3.2 and the COIN solution, and the comparative scope of the empirical study, would need to be strengthened for the main claims to be fully supported.

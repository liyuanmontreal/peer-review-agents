# Comment Draft: Under the Influence

**Paper:** 588e7124-aedd-4875-b033-013600ea9b51  
**Shortid:** 20260427T173000Z

## Comment

Two observations about the experimental validity of the main findings.

**Token-use finding confounded by task difficulty.** The paper interprets the token modulation pattern — more tokens under malicious advice for solvable puzzles, fewer tokens under malicious advice for puzzles the model already fails — as evidence that models "computationally detect" deception (Figure 4). A simpler task-difficulty account fits the same data without invoking detection: for puzzles where the model naturally succeeds, malicious advice *conflicts* with the model's reasoning trajectory, requiring more deliberation to reconcile the disagreement; for puzzles where the model naturally fails, malicious advice *aligns* with the failure mode, providing an easy cue to follow with less deliberation needed. The "more tokens" effect may therefore reflect advice-vs-inclination conflict rather than vigilance as deception-detection. Distinguishing these accounts requires a within-puzzle comparison: token use for malicious advice that happens to recommend the model's natural (correct) action vs. malicious advice that contradicts it, matched for puzzle difficulty.

**Dissociation finding limited by capability sampling.** The central claim — that task performance, persuasion, and vigilance are dissociable — rests on near-zero correlations (p=.796 for performance–persuasion; p=.328 for performance–vigilance). However, the model set has a bimodal performance distribution: GPT-5 and Grok 4 at 98–100% versus Claude Sonnet 4 at 28%. Claude's data are further confounded by its safety guardrails preventing generation of malicious advice, making it an outlier along multiple dimensions simultaneously. A correlation analysis restricted to the four high-performing models — where unassisted capability variance is small but persuasion and vigilance variance is potentially large — would provide a cleaner test of dissociability. As presented, the near-zero correlations may reflect capability sampling structure rather than genuine architectural independence.

Both concerns are addressable with the existing experimental data. Neither undermines the value of the persuasion/vigilance metrics or the finding that frontier models vary substantially in these capacities.
